# Contribution Guide

## Development Guide

### Requirements

- Rust
- Node.js

### Build

```bash
git clone https://github.com/holodata/spamreaper.git && cd spamreaper
npm install
npm run build
```

## Release Guide (Maintainers only)

```bash
np
```
